# Sample Event Runbook

## Event

AI/BI Intelligence Challenge

## Duration

2 hours

## Roles

- Host: leads event and debrief
- Facilitators: support teams without giving direct answers
- Ops: monitors app, validators, resources

## Timeline

| Time | Activity |
|---:|---|
| T-45 | Open host console and run dry-run |
| T-30 | Confirm teams and participant access |
| T-15 | Start lobby music / holding slide |
| T+0 | Welcome and rules |
| T+10 | Start event |
| T+40 | First leaderboard callout |
| T+80 | Bonus quest unlock |
| T+110 | 10-minute warning |
| T+120 | Freeze event |
| T+125 | Final leaderboard |
| T+135 | Debrief best practices |
| T+150 | Export report and close |

## Host script

```text
This is not a step-by-step workshop. Your team will be given a business problem and a set of quests. You can solve them however you choose, but the platform will validate the outcomes. Hints are available, but they cost points.
```

## Support rules

- Ask diagnostic questions first.
- Point participants toward hints.
- Do not solve the task for a team.
- Escalate broken validators to ops.
- Use manual adjudication only when necessary.

## End-of-event checklist

- Freeze event.
- Confirm pending validations complete.
- Apply manual adjustments.
- Export report.
- Save leaderboard screenshot.
- Reset or archive resources.
