# PR09 Prompt — Sample GameDay Packs

## Branch

`feature/gameday-pr09-sample-packs`

## Goal

Ship useful built-in quest packs that prove the platform.

## Requirements

Create at least two packs:

1. `ai_bi_gameday.yml`
2. `lakehouse_foundations.yml`

Each pack must include:

- narrative
- learning objectives
- resources
- at least 3 quests
- at least 6 tasks
- SQL validators
- at least one Databricks SDK validator if available
- hints
- facilitator notes

## Acceptance criteria

- Packs lint cleanly.
- Packs import cleanly.
- At least one event can be run end-to-end from a sample pack.
- Documentation explains how to customize packs.

## Verification

Run pack lint/import and dry-run validators.
