# PR06 Prompt — Admin Host Console

## Branch

`feature/gameday-pr06-host-console`

## Goal

Add a host console that lets facilitators run the event.

## Requirements

1. Add host event dashboard.
2. Add event lifecycle controls.
3. Add team/participant table.
4. Add validation queue/results view.
5. Add failed attempts view.
6. Add announcement composer.
7. Add manual score adjustment UI/API integration.
8. Add quest pack import/lint UI if not already implemented.

## Acceptance criteria

- Host can start/pause/freeze/complete event.
- Host can see teams and progress.
- Host can inspect validation attempts.
- Host can send announcements.
- Host can manually adjust score with reason.
- Player cannot see host console.

## Verification

- Build frontend.
- Manually test host and player roles.
